# Utterlane 使用说明

## 简介

Utterlane 是一款专注于「按句录音 + 时间轴拼接 + 字幕导出」工作流的录音软件。
适合配音、有声书、播客剪辑等需要把脚本拆成短句、逐句录制并允许多录次备选的场景。

## 入门

### 创建 / 打开工程

- **文件 → 新建工程**：弹出目录选择器，选定一个空文件夹,Utterlane 会在里面建立工程结构。
- **文件 → 打开工程**：选已有工程目录。
- 欢迎页也会列出最近打开过的工程,单击直接打开。

工程结构:每个工程是一个目录,包含 `project.json`(元信息)、`segments.json`(句段 + 录次列表) 和 `audios/` 子目录(实际 WAV 文件)。

### 导入脚本

- **文件 → 导入脚本…**:可以从纯文本 / 字幕文件导入大段文本,按行 / 段落自动拆成多个句段。

## 核心概念

- **句段(Segment)**:脚本中的一行或一段,是录音的基本单位。
- **录次(Take)**:每个句段下可以有多次录音;其中一个被标记为「当前录次」,参与最终拼接 / 导出。
- **节选(Trim)**:每个录次可以设置非破坏性的起 / 终点,原始 WAV 不动,导出时只用节选段。
- **尾部间隔(Trailing gap)**:两个句段之间的静音时长,导出时插入。

## 录音

1. 在 **句段** 面板选中要录的句子。
2. 在 **检查器** 顶部按下「录音」按钮(或快捷键 `R`)。
3. 倒计时(在偏好里设秒数)后开始录音。
4. 再按一次按钮停止 → 录次自动保存到工程。
5. 想要再录一遍同一句,按「重录」(`Shift+R`)。

录音过程中可以看到电平条:建议保持在 -12 ~ -6 dBFS 之间,避免削波。

## 录次管理

检查器底部表格显示当前句段的所有录次:

- **启用** 列勾选切换「当前录次」。
- **起点 / 终点(ms)** 直接编辑节选区间;波形面板会同步显示拖把。
- **试听 ▶**:单录次试听,不会改变当前选中。
- **删除 🗑** / **打开文件 📂**。

也可以在句段时间轴面板的波形上拖动起 / 终点把手做节选。

## 时间轴

两个面板,都支持鼠标滚轮 + 缩放滑块:

- **句段时间轴**:当前句段的细节视图,调节选 + 试听单句。
- **项目时间轴**:整个工程的横向时间轴,看到所有 clip 的顺序、时长、间隔;可以拖 clip 重排,拖间隔改变默认空白。

滚轮:
- 普通滚动 = 横向移动时间轴
- `Ctrl+滚轮` = 缩放横轴
- 在句段时间轴还支持 `Ctrl+Shift+滚轮` = 缩放纵轴(波形振幅)

## 导出

- **文件 → 导出 → 音频(WAV)**:把所有句段按顺序拼成一个 WAV,应用每段的节选 + 尾部间隔。可以选择整体 / 分段导出,可选峰值标准化。
- **文件 → 导出 → 字幕(SRT)**:根据每个句段的实际时长生成 SRT 字幕。

## 偏好设置

`编辑 → 偏好设置`(或 `Ctrl+,`)。分页包括:

- **外观**:Dock 主题、界面缩放、界面语言、文本对齐。
- **编辑器配色**:预设主题 + 16 个 token 自定义覆盖。
- **新工程默认**:新建工程时使用的采样率 / 声道数。
- **录音**:输入设备、录音前倒计时秒数。
- **快捷键**:可自定义传输 / 导航类快捷键。

## 常用快捷键

| 操作 | 默认快捷键 |
|------|-----------|
| 录音 / 停止 | `R` |
| 重录当前录次 | `Shift+R` |
| 播放当前句 / 暂停 | `Space` |
| 播放项目 / 暂停 | `Shift+Space` |
| 上一段 / 下一段 | `↑` / `↓` |
| 停止 / 取消 | `Esc` |
| 撤销 / 重做 | `Ctrl+Z` / `Ctrl+Y` |
| 偏好设置 | `Ctrl+,` |
| 查找 / 替换 | `Ctrl+F` |
| 界面缩放 | `Ctrl+=` / `Ctrl+-` / `Ctrl+0` |

## 数据安全

- 所有录音文件落在工程目录下的 `audios/`,原始 WAV 不会被节选 / 导出修改。
- 删除录次只是从 `segments.json` 中移除引用,文件保留在磁盘——可以通过「文件 → 音频审计」找回或彻底清理。
- 工程内容(segments.json)有撤销栈,绝大多数误操作可 `Ctrl+Z` 恢复。
